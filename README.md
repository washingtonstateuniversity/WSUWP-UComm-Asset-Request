ucomm.wsu.edu
=============

A WordPress child template for ucomm.wsu.edu. Depends upon the WSU Spine parent theme.

## Setup for WSUWP Indie Development

```
wsuwp-indie-sites:
  ucomm.wsu.edu:
    directory: ucomm.wsu.edu
    database: ucomm_wsu
    nginx:
      server_name: dev.ucomm.wsu.edu
      config: manual
```
